## Get languages

This endpoint allows you to get all the languages defined by the app params

**URL** : `/languages`

**Method** : <img src="https://img.shields.io/badge/GET%20-%23323330.svg?&style=flat&color=green"/>

**Auth required** : No

**Permissions required** : No

**Query parameters**

```
None
```
